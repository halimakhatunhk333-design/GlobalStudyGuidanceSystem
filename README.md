# GSGS (Global Study Guidance System)

## Project Description
GSGS is a database-driven guidance platform built via the LAMP stack (HTML/CSS/JS, PHP, MySQL). The PHP backend manages user authentication for the public dashboard, registration, and login. It executes secure SQL queries to dynamically process data for university search, course finder, SOP-builder, scholarship, financial guidance, visa guide, and career support modules.

## Objectives
* Develop a centralized global study guidance platform for international students.
* Simplify student registration and profile creation.
* Enable efficient university and course exploration through dynamic filtering.
* Provide an interactive tool for SOP building and career support mapping.
* Streamline access to scholarship resources and financial guidance.
